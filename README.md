# (s)mokeMoke — Midnight Shisha Cafe

深夜だけ開くシーシャカフェを舞台にした、会話と調合のアドベンチャーゲームです。

現在のバージョンは **Ver. 0.9.0** です。

## ブラウザ版

<https://yonkonken.github.io/smokemoke-mobile/>

インストールせず、PC・スマートフォンのブラウザから無料でプレイできます。
スマートフォンでは横画面でのプレイを推奨します。

> [!NOTE]
> 現在は友人向けの限定的な案内を想定していますが、GitHub Pages自体に閲覧制限やパスワードはありません。URLを知っている人はアクセスできます。

## ゲームの進め方

会話を読みながら物語を進め、来店客の注文に合わせてドリンクやシーシャを調合します。

- 会話画面をクリック／タップ：次の文章へ進む
- 文字送り中にクリック／タップ：文章を一括表示
- `Enter`／`Space`：次の文章へ進む
- 右クリック／`H`キー：画面内UIの表示・非表示
- 右上「設定」：BGM・SE・文字送り速度の変更、ロード、タイトルへ戻る
- 右上「LOG」：会話ログを表示
- 右上「レシピ」：発見済みレシピを表示
- 右下の目のボタン：画面内UIを隠す
- 右下の早送りボタン：既読文章をスキップ
- 右下の音符ボタン：ミュージックプレイヤーを開く

### 調合

1. ベースを1つ選びます。
2. アクセントを2つ選びます。
3. フレーバープロフィールを確認します。
4. 「作る」を押して提供します。

小さい画面では、右側の素材一覧を上下にスクロールできます。

## 対応環境

### PC

- Windows 10／11
- Google Chrome、Microsoft Edgeなどの最新版ブラウザ
- 1280×720以上、16:9前後の横長画面を推奨

### モバイル

- iPhone：Safari最新版
- Android：Google Chrome最新版
- 横画面を推奨

動作確認はiPhone SafariおよびLDPlayer上のAndroid Chromeで行っています。
Androidエミュレーターでの結果は、すべてのAndroid実機での動作を保証するものではありません。

初回表示やBGM再生には通信が必要です。プライベートブラウズ／シークレットモードでは、セーブデータが正常に保持されない場合があります。

## スマートフォンで遊ぶ

### iPhone／iPad

1. Safariでブラウザ版のURLを開きます。
2. 共有ボタンを押します。
3. 「ホーム画面に追加」を選びます。
4. 追加された `(s)mokeMoke` のアイコンから起動します。
5. 端末を横向きにしてプレイします。

Safariのタブから直接遊ぶこともできます。音声は、最初のタップ後に再生が始まります。

### Android

1. Chromeでブラウザ版のURLを開きます。
2. 右上のメニュー（︙）を開きます。
3. 「ホーム画面に追加」または「アプリをインストール」を選びます。
4. ホーム画面のアイコンから起動します。
5. 端末を横向きにしてプレイします。

端末やChromeのバージョンにより、メニュー名が少し異なる場合があります。

## セーブデータ

セーブデータはサーバーではなく、プレイしたブラウザ／端末内に保存されます。アカウント連携や端末間同期には対応していません。

- 各DAY開始時に自動セーブ
- タイトル画面の「つづきから」で、到達済みのDAYを選択可能
- 設定画面の「ロードする」からもDAYを選択可能
- 既読情報、ルート進行、発見済みレシピ、CG、音量設定、お気に入りBGMも端末内に保存

ブラウザ版とホーム画面へ追加した版で、保存領域の扱いが異なる場合があります。同じ端末でも、別ブラウザではセーブデータを共有できません。

### 「はじめから」について

「はじめから」を選ぶと、物語の進行状況とDAY別セーブを最初から開始します。
発見済みレシピ、CG、既読情報、音量設定などは残ります。

### セーブデータをすべて削除する

完全に初期化する場合は、ブラウザまたはOSの設定から、このサイトの保存データを削除してください。この操作は取り消せません。

- PC Chrome／Edge：アドレスバー左側のサイト情報から「サイトの設定」を開き、保存データを削除
- Android Chrome：「設定」→「サイトの設定」→「すべてのサイト」から `yonkonken.github.io` を選び、データを削除
- iPhone Safari：「設定」→「Safari」→「詳細」→「Webサイトデータ」から `github.io` のデータを削除
- ホーム画面版：アプリの削除後もデータが残る場合は、上記のWebサイトデータも削除

OSやブラウザのバージョンによって、項目名や場所が異なる場合があります。

## 音声について

- BGM・SEは設定画面からON／OFFおよび段階的な音量変更ができます。
- スマートフォン版では、文字送り音を無効にしています。
- ブラウザやOSの制限により、最初のタップまでは音声が再生されません。
- アプリをバックグラウンドへ移動した場合、BGMは一時停止します。

## 更新時の注意

通常のページ更新ではセーブデータは消えません。
表示が古い場合はページを再読み込みしてください。ホーム画面版で更新が反映されない場合は、一度終了してから再起動してください。

サイトデータを削除するとセーブデータも消えるため、キャッシュ削除時は対象を確認してください。

## 既知の制限

- 縦画面でのプレイには対応していません。
- オフラインプレイは保証していません。
- セーブデータの書き出し、読み込み、クラウド同期には未対応です。
- 端末性能やブラウザによって、画像表示や音声再生に時間がかかる場合があります。
